# Booklist for Psychology and Neuroscience

### Before reading:
I know this is a weird place to put a booklist, but I noticed that this is actually pretty convenient, so here it is! This is for anyone who need to locate a book for psychology related study. BTW, this is a CAS project for service, so if anyone benefitted from this, please put a comment! Thanks a lot. Give suggestions as well!

### What's Inside:
Inside this booklist, I will put all kinds of psychology and neuroscience related books. I will sort them by their content, such as "Developmental Theory", or "Animal Experiments" or something like that (P.S. I am not sure if the two mentioned will be added in, it is just an example). I hope that it will help people reach the books they want to conviniently. For some books, I will try to add a brief summary, for others in which I do not have time to write the summary, I will only have the name inputted.
